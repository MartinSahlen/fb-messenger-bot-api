Class FacebookMessagingAPIClient

Class representing an Send API client for Facebook. Can leverage existing classes/builders in project to provided required structured objects for method payloads. Can handle proxy by specifying 2nd argument as proxy to constructor Can send image/video/audio/file payload as both url and reference ID
